# URBAN STYLE - Tienda de Ropa

Página web básica de una tienda de ropa (solo diseño en HTML y CSS, no funcional).

## Páginas

- **index.html** — Tienda principal con productos destacados y categorías
- **login.html** — Pantalla de inicio de sesión
- **admin.html** — Panel de administración con dashboard, productos y pedidos

## Tecnologías

- HTML5
- CSS3 (sin frameworks)

## Cómo verlo

Abrir `index.html` en cualquier navegador.
